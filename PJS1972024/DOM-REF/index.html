Ejercicio 1: Cambio de Estilo Dinámico

Crea una página con tres botones: "Rojo", "Verde" y "Azul". Al hacer clic en cada botón, cambia el color de fondo de un elemento de la página a rojo, verde o azul, respectivamente.

Cambio de color

Ejercicio 2: Creación de Elementos

Diseña una lista de tareas pendientes. Agrega un campo de entrada y un botón "Agregar Tarea". Al hacer clic en el botón, crea dinámicamente un nuevo elemento de lista (por ejemplo, un <li>) con el contenido del campo de entrada y agrégalo a la lista.

Ejercicio 3: Eliminación de Elementos

Crea una lista de imágenes. Al hacer clic en una imagen, elimínala dinámicamente de la lista.

Ejercicio 4: Evento de Teclado

Crea un cuadro de texto que muestre dinámicamente el texto ingresado en un elemento de párrafo a medida que se escribe.

Ejercicio 5: Cambio de Clase CSS

Crea una galería de imágenes. Al hacer clic en una imagen, cambia dinámicamente su estilo aplicándole una clase CSS que destaque la imagen seleccionada.

Ejercicio 6: Animación de Elemento

Crea un botón que, al hacer clic, desplace suavemente un elemento de la página a una posición diferente utilizando una animación.

Ejercicio 7: Obtención de Atributos

Diseña una galería de imágenes con miniaturas. Al hacer clic en una miniatura, muestra dinámicamente la imagen completa en un área designada y muestra una alerta con la descripción de la imagen obtenida de un atributo personalizado.

Ejercicio 8: Contador de Clicks

Crea un juego sencillo donde un botón cuenta los clics realizados. Muestra dinámicamente el número de clics y felicita al jugador cuando alcanza cierta cantidad.
